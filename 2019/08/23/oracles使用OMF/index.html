oracles使用OMF

安装oracle后，使用命令可以使用OMF
ALTER SYSTEM SET DB_CREATE_FILE_DEST = ‘/u01/oradata’;

Once DB_CREATE_FILE_DEST is set, the DATAFILE clause can be omitted from a CREATE TABLESPACE statement. The datafile is created in the location specified by DB_CREATE_FILE_DESTby default. For example:
SQL> CREATE TABLESPACE tbs_2;
